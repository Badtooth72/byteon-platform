Explanation

Use input() to read user text (a str). Convert with int() or float() for maths. Use print() to display results. Add end='' to avoid a newline and sep=' ' to control spacing.

Further reading: W3Schools

Worked example
name = input("Name: ")
age = int(input("Age: "))
print(f"Hi {name}! In 5 years you'll be {age + 5}.")
Line-by-line: We cast age to int so we can add 5. f-strings combine text and variables neatly.
