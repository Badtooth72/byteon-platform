Loops

Repeat actions with for and while.

Explanation

Loops repeat actions. Use for when you know how many times; while when you repeat until a condition changes.

Example
for i in range(1, 6):
    print("For loop i =", i)

n = 5
while n > 0:
    print("While loop n =", n)
    n -= 1
Easy task

Print all the numbers from 1 to 20 on one line separated by spaces.

Harder task

Ask for a number n and print the times table for n up to 12×.
