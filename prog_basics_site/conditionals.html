Conditionals

If your program needs to choose, use if/elif/else.

Explanation

Conditionals choose between paths. Use if, elif, else with boolean expressions. Indentation defines the blocks that belong to each condition.

Further reading: W3Schools

Worked example
score = int(input("Score (0-100): "))
if score >= 80:
    grade = "A"
elif score >= 60:
    grade = "B"
else:
    grade = "C"
print("Grade:", grade)
Line-by-line: Only one branch runs: first True condition wins. The final else runs if all above are False.
Easy task

Ask for a number. Print 'even' if divisible by 2, else 'odd'.

  • Variables: n
  • Technique: selection
Example output: Input: 6 → even
Harder task

Ask for a year and state if it's a leap year (divisible by 4 and not by 100 unless also by 400).

  • Variables: year
  • Technique: selection + logical operators
Example output: Input: 2000 → leap year
