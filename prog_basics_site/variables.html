Variables

Store and name values to reuse later.

Explanation

Variables name values in memory. Created when you assign (using =). Names can contain letters, digits, and underscores, but cannot start with a digit.

Further reading: W3Schools

Worked example
name = "Jordan"
age = 14
height_m = 1.68
is_student = True
bmi = round(60 / (height_m ** 2), 1)
print(f"{name} is {age} years old. BMI {bmi}")
Line-by-line: Four variables of different types: str, int, float, bool. BMI is computed with exponentiation and then rounded; an f-string formats the message.
Easy task

Create variables first_name, year_group, favourite_subject. Print a sentence using all three.

  • Variables: first_name, year_group, favourite_subject
  • Technique: sequence
Example output: My name is Sam in Year 9 and I enjoy Computing.
Harder task

Create width and height and calculate area and perimeter of a rectangle. Output both with labels using an f-string.

  • Variables: width, height, area, perimeter
  • Technique: sequence + arithmetic
Example output: Area: 24 Perimeter: 20
