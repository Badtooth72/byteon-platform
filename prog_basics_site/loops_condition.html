Condition-controlled loops (while)

Repeat until a condition becomes False.

Explanation

Condition-controlled iteration repeats while a condition is True. Ensure the loop changes something, otherwise it can run forever.

Further reading: W3Schools

Worked example
n = 5
while n > 0:
    print(n)
    n -= 1
print("Blast off!")
Line-by-line: n decreases towards zero; loop stops when the condition becomes False.
Easy task

Ask for a password until the user types 'secret'. Then print 'Unlocked'.

  • Variables: pwd
  • Technique: iteration (while) + selection
Example output: Try again... Try again... Unlocked
Harder task

Keep asking for numbers until 0 is entered; then print the total and average.

  • Variables: x, total, count
  • Technique: iteration (while) + accumulator
Example output: Total: 42 Average: 8.4
