Explanation

Functions group steps under a name. Parameters pass data in; return sends a value back. Good for reuse and testing.

Further reading: W3Schools

Worked example
def greet(name):
    print(f"Hello, {name}!")

def add(a, b):
    return a + b

greet("Alex")
print(add(2, 3))
Line-by-line: One function prints, the other returns a sum that we then print.
Easy task

Write a function shout(msg) that prints the message in uppercase.

  • Variables: msg
  • Technique: sequence + function definition
Example output: shout('hi') → HI
Harder task

Write bmi(weight, height) that returns BMI rounded to 1 dp, then use it in a sentence.

  • Variables: weight, height, bmi_value
  • Technique: sequence + return value
Example output: BMI 22.1
