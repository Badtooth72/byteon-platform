Count-controlled loops (for)

Repeat a fixed number of times.

Explanation

Count-controlled iteration repeats a known number of times. Use for with range(start, stop[, step]). stop is exclusive.

Further reading: W3Schools

Worked example
for i in range(1, 6):
print("i =", i)
Line-by-line: range(1,6) yields 1..5. The body runs once per value of i.
Easy task

Print numbers 1 to 20 on one line separated by spaces.

  • Variables: i
  • Technique: iteration (for)
Example output: 1 2 3 4 ... 20
Harder task

Ask for a number n and print its times table up to 12×.

  • Variables: n, i
  • Technique: iteration (for) + input
Example output: 7 × 1 = 7 ... 7 × 12 = 84
