Lists (1D)

Add, remove, update, and insert items.

Explanation

Lists hold ordered collections. Create with []. append() adds, insert() positions, remove() deletes by value, pop() removes and returns by index (default last), and indexing updates items.

Further reading: W3Schools

Worked example
nums = [5, 2, 9]
nums.append(7)       # add to end
nums.insert(1, 6)    # insert 6 at index 1
nums[0] = 10         # update first element
nums.remove(9)       # remove by value
last = nums.pop()    # remove and return last
print(nums, "last removed:", last)
Line-by-line: We show add, insert, update, remove, and pop; print shows the final list and the removed value.
