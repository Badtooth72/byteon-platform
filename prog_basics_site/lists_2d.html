2D Lists

Tables and grids using lists of lists.

Explanation

2D lists are lists of lists, useful for tables or grids. Access items with two indices: list[row][col].

Further reading: W3Schools

Worked example
marks = [
    ["Alex", 7, 6, 8],
    ["Bea",  9, 7, 8],
    ["Cam",  6, 5, 7]
]
marks[1][2] = 8  # update Bea's second mark
print(marks)
Line-by-line: We build a small table then update a cell at row 1, col 2.
Easy task

Create a 2×3 grid of zeros and then set the value in row 1, col 1 to 5.

  • Variables: grid
  • Technique: nested lists + indexing
Example output: [[0,0,0],[0,5,0]]
Harder task

Store a small timetable (2 students × 3 lessons). Print each student's name with their average.

  • Variables: table, i
  • Technique: nested iteration + arithmetic
Example output: Alex → 7.0, Bea → 8.0
